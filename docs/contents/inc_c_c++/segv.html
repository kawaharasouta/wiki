segv対策¶

gdb¶

コンパイル時にデバッグ情報を付与¶

$ gcc -g ***.c

or Makefileに

CFLAGS += -g

とかしておく

コアダンプ機能を有効化¶

ulimit -c unlimited

デバッグファイル¶

問題のプログラムを実行すると

$ ls
...
core

coreとかいうでっかいファイルができる

コマンド¶

バックトレース
(gdb) bt

関数選択
frame 1

ポインタ検索
